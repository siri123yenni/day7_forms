HTML:
HTML stands for HyperText Markup Language. It is used to create the structure of web pages. HTML uses tags to display text, images, links, and other content.
TAGS:
Tags are keywords enclosed in angle brackets. They tell the browser how to display content. Most tags have opening and closing tags.
ATTRIBUTES:
Attributes give additional information about tags. They are written inside the opening tag. Example attributes are id, class, href, src.
MARQUEE
tag is used to show scrolling text or images. The text can move left, right, up, or down. This tag is deprecated in HTML5.
TEXT FORMATTING:
Text formatting tags change the appearance of text. They are used to make text bold, italic, or emphasized.
ANCHOR TAG:
The anchor tag is used to create hyperlinks. It helps to connect one page to another page or site. The href attribute specifies the link address.
